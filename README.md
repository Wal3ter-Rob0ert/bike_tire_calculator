# Bike Tire Pressure Calculator

## Project Overview
This project is a simple interactive web application that helps cyclists estimate the recommended tire pressure for their bicycle tires.  
The calculator takes into account several parameters such as:

- Rider weight  
- Tire width  
- Bike type (Road, Gravel, MTB)  
- Riding surface  

Based on these inputs, the application calculates recommended pressure values for:

- Front tire
- Rear tire

The goal of this project is to demonstrate the use of **basic web development technologies** including HTML, CSS, and JavaScript.

---

## Technologies Used

- **HTML** – for the structure of the webpage  
- **CSS** – for styling, layout, and visual design  
- **JavaScript** – for interactivity and pressure calculations  

The project runs entirely in the browser and **does not require a backend**.

---

## Features

- Bike type selection (Road / Gravel / MTB)
- Rider weight slider
- Tire width slider
- Surface type selection
- Pressure calculation for front and rear tires
- Pressure level indicator (Low / Optimal / High)
- Warning messages for unusual pressure values
- Random cycling tips
- Dark mode toggle
